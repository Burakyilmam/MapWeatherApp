<h1>Haritalı Hava Durumu Uygulaması</h1>

<p>Türkiye genelindeki illerin hava durumu verilerini (sıcaklık, rüzgar, yağış vb.) interaktif bir harita üzerinde görselleştiren; kullanıcı etkileşimiyle şehir detaylarını, hava durumu bilgilerini dinamik olarak sunan modern bir web uygulamasıdır.</p>

<h3>Kullanılan Teknolojiler</h3>
<ul>
  <li><b>HTML</b></li>
  <li><b>CSS</b></li>
  <li><b>JavaScript</b></li>
  <li><b>Bootstrap</b></li>
  <li><b>ASP.NET Core MVC</b></li>
  <li><b>ASP.NET Core Web API</b></li>
  <li><b>Entity Framework Core</b></li>
  <li><b>Microsoft SQL Server (MSSQL)</b></li>
  <li><b>Leaflet.js</b></li>
  <li><b>GeoJSON</b></li>
  <li><b>SweetAlert2</b></li>
  <li><b>Chart.js</b></li>
  <li><b>OpenWeatherMap API</b></li>
</ul>

<h3>Geliştirme Süreci</h3>
<p>Proje geliştirme sürecinde AI destekli araçlardan (ChatGPT , Gemini vb.) faydalanılmıştır.</p>

<h3>Özellikler</h3>
<ul>
  <li>Türkiye şehirlerinin GeoJSON formatında harita üzerinde dinamik olarak sınırlarının gösterilmesi</li>
  
  <li>OpenWeatherMap API ile gerçek zamanlı hava durumu verisi entegrasyonu </li>

  <li>Şehirlere ait hava durumu verilerinin veritabanında saklanması ve son 30 dakikalık verilerin listelenmesi</li>
  
  <li>Sıcaklık , Rüzgar , Yağış / Nem / Kar / Bulut / Görüş Mesafesi / Basınç seçili katman değerlerine göre dinamik renk skalası ve harita görselleştirmesi</li>
  
  <li>Gelişmiş şehir arama sistemi (otomatik tamamlama + klavye ile gezinme desteği)</li>
  
  <li>Seçilen şehre odaklanma (harita animasyonu ile yakınlaşma)</li>
  
  <li>Günlük istatistik paneli (en sıcak, en soğuk, en rüzgarlı vb.)</li>
  
  <li>Tüm şehirler için detaylı veri tablosu ve etkileşimli seçim sistemi</li>
  
  <li>Farklı harita sağlayıcıları arasında dinamik geçiş</li>

  <li>Çoklu veri katmanı desteği (Sıcaklık / Rüzgar / Yağış / Nem / Kar / Bulut / Görüş Mesafesi / Basınç)</li>
  
  <li>Modern tasarıma sahip etkileşimli bilgi paneli ve tooltip sistemi</li>
  
  <li>Sürüklenebilir popup ve kontrol panelleri</li>

  <li>Harita kontrollerinin küçültülüp genişletilebilmesi</li>

  <li>Dinamik legend sistemi ve katmana göre otomatik gösterge güncellemesi</li>
</ul>
